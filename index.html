HTML Cheat Sheet

Etiquetas Básicas

<html>

Define el documento HTML.

<html>...</html>

<head>

Contiene metadatos y enlaces a recursos externos, como hojas de estilo.

<head></head>

<body>

Contiene el contenido visible del documento HTML.

<body></body>

Estructura Semántica

<header>

Define la cabecera de un documento o sección.

<header>...</header>

<nav>

Define una sección de navegación.

<nav>...</nav>

<main>

Contiene el contenido principal del documento.

<main>...</main>

<aside>

Define contenido adicional o complementario.

<aside>...</aside>

<footer>

Define el pie de página de un documento o sección.

<footer>...</footer>

Párrafos

<p>

Define un párrafo de texto.

<p>Este es un párrafo.</p>

Encabezados

<h1> - <h6>

Define los encabezados, desde el más importante (<h1>) hasta el menos importante (<h6>).

<h1>Encabezado 1</h1>
<h2>Encabezado 2</h2>
<h3>Encabezado 3</h3>
<h4>Encabezado 4</h4>
<h5>Encabezado 5</h5>
<h6>Encabezado 6</h6>

Enlaces

<a>

Define un hipervínculo.

<a href="https://www.adondesea/masalladelsol.com">Este es un enlace</a>

Listas

<ol>

Define una lista ordenada.

<ol><li>Item 1</li><li>Item 2</li></ol>

<ul>

Define una lista desordenada.

<ul><li>Item 1</li><li>Item 2</ul>

<dl>

Define una lista de definición.

<dl>
  <dt>Termino 1</dt>
  <dd>Definición 1</dd>
  <dt>Termino 2</dt>
  <dd>Definición 2</dd>
</dl>

<li>

Define un ítem en una lista.

<li>Item</li>

<dt>

Define un término en una lista de definición.

<dt>Término</dt>

<dd>

Define una descripción de un término en una lista de definición.

<dd>Descripción</dd>

Multimedia

<img>

Define una imagen.

<img src="imagen.jpg" alt="Descripción de la imagen">

<audio>

Define contenido de audio.

<audio controls><source src="audio.mp3" type="audio/mpeg"></audio>

<video>

Define contenido de vídeo.

<video controls><source src="video.mp4" type="video/mp4"></video>

<table>

<tr>

Define una fila en una tabla.

<tr>...</tr>

<td>

Define una celda en una tabla.

<td>...</td>

<th>

Define una celda de encabezado en una tabla.

<th>...</th>

Formularios

<form>

Define un formulario HTML.

<form>...</form>

<input>

Define un campo de entrada en un formulario.

<input type="text" name="nombre">

Variantes del atributo "type" en <input>

  • <input type="text"> Campo de texto.
  • <input type="number"> Campo numérico.
  • <input type="tel"> Campo de teléfono.
  • <input type="email"> Campo de email.
  • <input type="password"> Campo de contraseña.
  • <input type="radio"> Botón de opción.
  • <input type="checkbox"> Casilla de verificación.
  • <input type="button"> Botón.
  • <input type="color"> Selector de color.
  • <input type="date"> Selector de fecha.
  • <input type="time"> Selector de hora.
  • <input type="datetime-local"> Selector de fecha y hora.
  • <input type="file"> Campo para subir archivos.
  • <input type="hidden"> Campo oculto.

<select>

Define una lista desplegable.

<select name="opciones">
  <option value="1">Opción 1</option>
  <option value="2">Opción 2</option>
</select>

<textarea>

Define un área de texto.

<textarea name="comentarios">Escribe aquí...</textarea>

<label>

Define una etiqueta para un elemento de formulario.

<label for="nombre">Nombre:</label><input type="text" id="nombre" name="nombre">
